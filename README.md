# MOA TICKETS FRONTEND

## TECH STACKS

- React 19
- Typescript
- TailwindCSS
- React-icons

## PACKAGE DESIGN
```
src
  ├── app             # provider 설정 및 style 설정
  ├── entities        # 리듀서 설정 및 constant
  ├── features        # 커스텀 훅
  ├── pages           # 라우터 페이지
  ├── shared          # 공통 컴포넌트
  └── widgets         # 한번만 쓸 컴포넌트들
```
